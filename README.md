# Youtube-Downloader
Simple tools that lets you download videos from Youtube.

How to use -
1. Copy Youtube Video URL.
2. Paste Video URL.
3. Choose save loacation ( optional) - By Default video goes to Downloads Folder
4. Click Download.
5. Enjoy your Video :)
